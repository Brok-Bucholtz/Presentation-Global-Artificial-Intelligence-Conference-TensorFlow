# Global Artificial Intelligence Conference - TensorFlow Presentation

```sh
pip install -r requirements.txt
```

```

jupyter notebook notebooks/3_cnn_model.ipynb
```
### Transfer Learning
```sh
jupyter notebook notebooks/4_transfer_learning.ipynb
```
